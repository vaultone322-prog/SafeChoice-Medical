# Product Image File Names

Place all product images in the `/public/images/` folder with these exact filenames:

## Family Planning Products
- `sayana-press.jpg` - Sayana Press (Family Planning 3 Months)
- `lydia.jpg` - Lydia (Emergency Contraceptive)
- `p2.jpg` - P2 (Contraceptive Pill)
- `backup.jpg` - Backup (Emergency Contraceptive)
- `family-planning.jpg` - Family Planning (One Month Supply)

## Women's Health Products
- `cytotec.jpg` - Cytotec (Women's Health Medication)
- `ma-care.jpg` - Ma Care (Women's Wellness Support)

## Antibiotics
- `ampiclox.jpg` - Ampiclox (Broad-Spectrum Antibiotic)
- `amoxicillin.jpg` - Amoxicillin (Penicillin Antibiotic)
- `cipro.jpg` - Ciprofloxacin/Cipro (Fluoroquinolone Antibiotic)
- `doxycycline.jpg` - Doxycycline (Tetracycline Antibiotic)
- `metro.jpg` - Metronidazole/Metro (Antibiotic & Antiprotozoal)
- `cefixime.jpg` - Cefixime (Cephalosporin Antibiotic)

## Pain Relief
- `diclofenac.jpg` - Diclofenac (NSAID Pain Reliever)

## Men's Health
- `viagra.jpg` - Viagra (Erectile Dysfunction Treatment)

## Abortion Care Kits
- `complete-care-kit-standard.jpg` - Complete Care Kit - Standard
- `complete-care-kit-premium.jpg` - Complete Care Kit - Premium
- `essential-care-kit.jpg` - Essential Care Kit

## Optional
- `placeholder.jpg` - Fallback image if a product image is missing

## Notes
- Images should be in JPG format (you can use PNG if preferred, just update the extension in the code)
- Recommended size: 400x300px or larger (will be automatically resized)
- All images should be placed in `/public/images/` folder
- The code will automatically reference them as `/images/filename.jpg`
